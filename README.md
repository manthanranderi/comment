📩 Feedback Form Application

A simple and responsive Feedback Submission Form built using modern web technologies. This project focuses on form validation, user interaction, and clean UI design, making it ideal for beginners learning frontend development.

🚀 Features

🧾 User-friendly feedback form UI

✅ Real-time form validation

📧 Email format validation

📂 Category & Priority selection

📝 Issue description with minimum character validation

💾 Displays submission status

🎨 Clean and minimal design

🛠️ Technologies Used

HTML5 – Structure of the form

CSS3 – Styling and layout

JavaScript – Form validation and interactivity

📁 Project Structure
project-folder/
│── index.html
│── style.css
│── script.js
└── README.md
📸 Preview

The application includes:

Input fields for Name and Email

Dropdowns for Category and Priority

Textarea for issue description

Validation messages for incorrect inputs

Submit button with feedback status display

⚙️ How It Works

User fills out the form

JavaScript validates:

Required fields

Proper email format

Minimum 10 characters in description

If valid:

Form is submitted

Feedback message is shown

If invalid:

Error messages appear below fields

▶️ How to Run

Download or clone the repository

Open the project folder

Run index.html in your browser
